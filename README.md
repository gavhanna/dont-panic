## Installation
* Clone this repo `git clone https://github.com/gavhanna/dont-panic.git dontPanic`
* CD into directory `cd dontPanic`
* run `bundle install`
* run `rake db:seed`
* run `rake db:migrate`
* run `rails s -b $IP -p $PORT`
* Click the `Preview` tab and select `Preview running application`
